# Colors

SoliDS utilizza **color tokens semantici**, non colori grezzi. I valori dipendono dal **tema attivo** (`data-theme`): light, dark, fantasy e cyberpunk ridefiniscono la stessa struttura semantica.

I consumer **non devono mai** usare direttamente la palette base
(es. `gray.900`, `blue.600`), ma solo token con significato.

---

## Semantic color tokens

### Text

- `--sd-color-text-primary`
- `--sd-color-text-secondary`
- `--sd-color-text-tertiary`
- `--sd-color-text-disabled`
- `--sd-color-text-inverse`
- `--sd-color-text-link` / `--sd-color-text-link-hover`

### Background

- `--sd-color-bg-canvas`
- `--sd-color-bg-surface`
- `--sd-color-bg-elevated`
- `--sd-color-bg-overlay` / `hover` / `active` / `disabled`

### Border

- `--sd-color-border-default`
- `--sd-color-border-muted`
- `--sd-color-border-strong`
- `--sd-color-border-focus`
- `--sd-color-border-disabled`

### Intent

- `--sd-color-intent-success` (+ `-bg`, `-border`)
- `--sd-color-intent-warning` (+ `-bg`, `-border`)
- `--sd-color-intent-danger` (+ `-bg`, `-border`)
- `--sd-color-intent-info` (+ `-bg`, `-border`)

### Icon (stroke / `currentColor`)

Usate dalle icone SoliDS e dalle utility `.sd-icon`, `.sd-icon-muted`, â€¦

- `--sd-color-icon-default`
- `--sd-color-icon-muted`
- `--sd-color-icon-primary`
- `--sd-color-icon-on-primary`

### Componenti (brand)

- `--sd-color-primary-*`, `--sd-color-secondary-*`, `--sd-color-muted-*`, `--sd-color-accent-*`, `--sd-color-destructive-*`

export function ColorGrid() {
  const items = [
    { name: "Text / Primary", var: "--sd-color-text-primary" },
    { name: "Text / Secondary", var: "--sd-color-text-secondary" },
    { name: "Background / Canvas", var: "--sd-color-bg-canvas" },
    { name: "Background / Surface", var: "--sd-color-bg-surface" },
    { name: "Border / Default", var: "--sd-color-border-default" },
    { name: "Border / Focus", var: "--sd-color-border-focus" },
    { name: "Intent / Success", var: "--sd-color-intent-success" },
    { name: "Intent / Warning", var: "--sd-color-intent-warning" },
    { name: "Intent / Danger", var: "--sd-color-intent-danger" },
    { name: "Intent / Info", var: "--sd-color-intent-info" }
  ];

return (
<div
style={{
        display: "grid",
        gridTemplateColumns: "repeat(auto-fill, minmax(220px, 1fr))",
        gap: 12
      }} >
{items.map((item) => (
<div
key={item.var}
style={{
            border: "1px solid var(--sd-color-border-default)",
            borderRadius: 10,
            padding: 12,
            background: "var(--sd-color-bg-surface)",
            color: "var(--sd-color-text-primary)"
          }} >
<div style={{ fontSize: 12, opacity: 0.7 }}>{item.name}</div>
<div style={{ fontSize: 12, marginBottom: 8 }}>{item.var}</div>
<div
style={{
              height: 32,
              borderRadius: 6,
              background: `var(${item.var})`,
              border: "1px solid var(--sd-color-border-muted)"
            }}
/>
</div>
))}
</div>
);
}
