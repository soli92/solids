# Spacing

Lo spacing in SoliDS è basato su una **scala numerica** in `base.json` (es. `--sd-space-1` … `--sd-space-6`, fino a valori più grandi) e su **alias semantici** in `semantic.json` (es. `--sd-space-xs` … `--sd-space-3xl`) che mappano su passi della scala.

L’obiettivo è:

- ridurre variazioni arbitrarie
- migliorare l’allineamento
- rendere il layout prevedibile e scalabile

---

## Regole fondamentali

- ✅ Usa **solo** valori della scala (`var(--sd-space-*)` o alias `--sd-space-xs` …)
- ❌ Evita `px` arbitrari nei componenti
- ✅ Preferisci token semantici allo spacing “ad hoc”

---

## Scala di esempio (token numerici)

| Token | Uso tipico |
| ----- | ---------- |
| `--sd-space-0` | reset / collapse |
| `--sd-space-1` | gap minimo (4px) |
| `--sd-space-2` | padding interno stretto (8px) |
| `--sd-space-3` | separazione elementi (12px) |
| `--sd-space-4` | layout standard (16px) |
| `--sd-space-5` | respiro medio (20px) |
| `--sd-space-6` | separazioni più ampie (24px) |

La lista completa (fino a `--sd-space-64`, più `px` / frazioni) è in `dist/tokens/tokens.json` e in `dist/css/variables.css`.

export function SpacingScale() {
  const items = [
    { name: "space-0", var: "--sd-space-0" },
    { name: "space-1", var: "--sd-space-1" },
    { name: "space-2", var: "--sd-space-2" },
    { name: "space-3", var: "--sd-space-3" },
    { name: "space-4", var: "--sd-space-4" },
    { name: "space-5", var: "--sd-space-5" },
    { name: "space-6", var: "--sd-space-6" }
  ];

return (
<div style={{ display: "flex", flexDirection: "column", gap: 12 }}>
{items.map((item) => (
<div
key={item.var}
style={{
            display: "flex",
            alignItems: "center",
            gap: 12,
            color: "var(--sd-color-text-primary)"
          }} >
<div style={{ width: 120, fontSize: 12 }}>
{item.name}
<div style={{ opacity: 0.6 }}>{item.var}</div>
</div>

          <div
            style={{
              height: 16,
              width: `var(${item.var})`,
              minWidth: 1,
              background: "var(--sd-color-intent-info)",
              borderRadius: 4
            }}
          />

          <div style={{ fontSize: 12, opacity: 0.6 }}>
            var({item.var})
          </div>
        </div>
      ))}
    </div>

);
}
