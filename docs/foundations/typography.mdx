# Typography

La tipografia in SoliDS è definita tramite token semantici.

## Obiettivi

- Leggibilità
- Coerenza
- Scalabilità

## Principi

- Niente font-size hardcoded nei componenti
- Usare scale coerenti
- Separare dimensione, peso e line-height

## Esempi di utilizzo

- text.primary
- text.secondary

La tipografia può evolvere senza impattare i consumer.
