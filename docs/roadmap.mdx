# Roadmap

## Stato attuale

✅ Design tokens (base + semantic + temi **light**, **dark**, **fantasy**, **cyberpunk**)  
✅ CSS variables e pipeline `index.css`  
✅ Fallback OS dark senza `data-theme`  
✅ Base styles e utility CSS  
✅ Preset Tailwind e layer shadcn  
✅ Storybook: foundations aggiornate (temi, colori, spacing, tipografia, radius, token) + story UI

## Prossimi step

- Esempi e linee guida accessibilità più estese

## Visione

SoliDS vuole restare una base **solida, estendibile e manutenibile** per prodotti reali, con token e temi coerenti indipendentemente dallo stack UI.
