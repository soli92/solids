# SoliDS

**SoliDS** è un **Design System agnostico** basato su **design tokens** e **CSS**.  
Fornisce fondamenta riutilizzabili (colori, spacing, tipografia, temi, utility) consumabili da **qualsiasi framework** o progetto web.

Il pacchetto espone:

- ✅ Design tokens in formato JSON
- ✅ CSS variables semantic-first
- ✅ Supporto nativo a **light / dark theme**
- ✅ Base styles e utility CSS
- ✅ Un **entrypoint unico** (`index.css`) per un’integrazione semplice

---

## Installazione

```bash
npm install @Soli92/solids
```
